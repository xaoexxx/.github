# .github

GitHub organization configuration repository.

## EV (Eventual) CLI

This repository includes **EV**, a terminal-based chat/messaging application where conversations make things happen.

### Quick Start

```bash
# Show help
./ev help

# Try the example command
./ev +e ?xx(s) @node (runall) .repos

# Create events
./ev +e myevent

# Query conversations
./ev ?myevent
```

### Installation

```bash
./install.sh
```

For full documentation, see [EV-README.md](./EV-README.md).
